# WebLusBook
WebLusBook — это веб-платформа, предназначенная для того, чтобы покупать и прочитывать книги. Платформа позволяет пользователям искать подходящие книги , сортировать по рейтингу , а также делиться мнением о книге, оставляя комментарии.
| Раздел | Маршрут | Тип доступа | Описание |
|--------|--------|-------------|-----------|
| Главная | /     | Публичный   | Страница с книгами |
| Аутентификация | /auth/* | Публичный | Вход, регистрация , восстановление пароля |
| Каталог книг | /catalog | Публичный |  Список книг с фильтрацией по категориям, цене и т.д. |
| Страницв книги | /catalog/:id | Публичный | Полное описание книги ,возможность добавить в избранное |
| Поиск | /search | Публичный | Страница с результатом поиска |
| Личный кабинет | /account | Приватный | Личный кабинет пользователя с его данными |
| Избранное | /favorites | Приватный | Список понравившихся пользователю машин |
| Админ-панель | /admin | Приватный | Управление списком книг , учетными записяит пользователей и т.д. |
